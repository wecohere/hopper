    Hopper

    Hopper creates a consistent, codified interface between projects to help people get up-to-speed and contributing fast.

    Hopper will provide a common interface for project tasks across a variety of programming languages. Examples of common tasks:

    • Initializing a new project.
    • Navigating to the projects on your file system.
    • Running a project
    • Testing a project
    • Generating documentation
    • Building a package from a project
    • Pushing a packaged project to an artifact repository
    • Bumping a version number while following semver
    • Releasing a new version of a project

    Installation

    Once you have node installed, install Hopper with npm install -g hopper-cli.
